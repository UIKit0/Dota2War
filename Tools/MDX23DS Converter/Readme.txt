MDX Converter v0.0.1 (Alpha) by KMK (jimmi_pgl@hotmail.com)

This converter turns MDX files into 3DS files.
Pros:
    Flawless geometry and texture coordinates.
Cons:
    No animation.
    Textures:
        Need to be added manually.
    Overwrites without warning:
        If loaded file is X:\Path\file.mdx, converted file is X:\Path\file.3ds.

v0.0.1
    Textures map correctly (do not require flipping).
